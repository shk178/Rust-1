- 한 줄 주석
```rust
fn main() {
    // 주석
    let some_number = 100; // 주석
    // 주석
}
```
- 코드 중간 주석, 한 줄 이상 주석
```rust
fn main() {
    let some_number/*: i16*/ = 100;
}
```
- 함수, 구조체, enum, 모듈 같은 개별 항목 위에 작성
```rust
/// 이 함수는 두 수를 더합니다.
///
/// # 예시
/// ```
/// let sum = add(2, 3);
/// assert_eq!(sum, 5);
/// ```
fn add(a: i32, b: i32) -> i32 {
    a + b
}
```
- 소스 파일이나 모듈의 최상단에 작성
```rust
//! 이 모듈은 수학 관련 유틸리티를 제공합니다.
//!
//! # 사용법
//! ```
//! let result = math_utils::add(1, 2);
//! ```

pub fn add(a: i32, b: i32) -> i32 {
    a + b
}
```
